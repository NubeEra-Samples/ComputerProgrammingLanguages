Check git:-
-----------
git --version

One Time Configuration:-
------------------------
git init
git remote add origin https://github.com/NubeEra-Samples/FSD-Practicals.git
git config user.name "YOUR_GITHUB_USER_NAME"
git config user.email "YOUR_GITHUB_USER_EMAIL"


Everytime:-
-----------
git add .
git commit -m "MSG"
git push -u origin master
